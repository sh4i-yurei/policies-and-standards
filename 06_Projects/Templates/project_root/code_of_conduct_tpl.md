# Purpose

Define behavior expectations and how to report issues.

# Scope

Applies to all contributors, maintainers, and AI-assisted agents working
in this project.

# Standard

## Code of conduct template

Link to the maintainer contact and incident process.

## Expected behavior

- Be respectful and professional.
- Assume good intent; seek clarification before escalating.
- Follow project workflows, coding standards, and review rules.
- Give constructive feedback focused on work, not individuals.

## Unacceptable behavior

- Harassment, discrimination, or personal attacks.
- Unapproved disclosure of confidential or sensitive information,
  including logs or data with secrets, personal data, or protected info
  outside approved channels.
- Bypassing governance, review, or security controls.
